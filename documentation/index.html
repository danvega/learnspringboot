HTML Structure General layout and content structure

Studio is a multi purpose theme that is great for anything from creative portfolios to corporate websites. There are 5 main sections I want to summarize here. There is the navigation, the services section, the features section, the portfolio section, and the meet the team section.

Navigation

The navigation is a sidebar navigation made up of a Logo and Menu items. The menu items can be set up for smooth scroll or dropdown lists.

To make dropdown menus add the class dropdown to the list item and add a dropdown menu ul tag.

Bootstrap Columns

Bootstrap columns are the easiest way to make things responsive. just define a row div, and you can add columns to them. with tags like col-sm-4, that means you can fit 3 of those into a column on a small device. There are 12 columns in a row. Usually you want to define col-xs-12 as 12 columns because on small screens things will shrink.

<div class="row">
  <div class="col-md-4 col-sm-4 col-xs-12">
</div>

JavaScript JavaScript customization and highlights

× Bootstrap Javascript: Bootstrap comes with its own javascript file that you should never touch or change. It contains all the scripts for the bootstrap framework. I have also included that minimized version.

JavaScript Files

Only edit init.js, the rest of the files are plugins or libraries that shouldn't be edited

Tag Description
init.js This is the custom javascript that initializes everything like the carousels, web elements and more.
jquery-1.9.1.min.js jQuery is a Javascript library that greatly reduces the amount of code that you must write.
bootstrap.js This is the Bootstrap framework javascript that enables all of its functionalities. Do not edit this.
jquery.mixitup.js This is jQuery plugin that enables the portfolio sorting and categorization.
masonry.pkgd.min.js This is jQuery plugin that enables the portfolio sorting and categorization.
masonry-init.js This initializes the code for masonry. This is separate from the main init file because not all pages need masonry.
respond.min.js This enables css media queries in internet explorer browsers 8 and below. It is simply for cross browser compatibility.

Masonry Blog

Masonry is a plugin that allows things in a row to fit nicely together without unnecessary whitespace.

Here is the initializing code. If you wanna customize any masonry options it would go inside the $container.masonry({ block.

  var $container = $('.masonry-grid');
  
  $container.imagesLoaded(function(){
    new AnimOnScroll( document.getElementById( 'grid' ), {
      minDuration : 0.4,
      maxDuration : 0.7,
      viewportFactor : 0.2
    } );
  });

Google Maps

Take a look at the maps api for better documentation.

<div id="map-canvas"></div>
<script type="text/javascript">
function initialize() {
  var myLatLong = new google.maps.LatLng(40.7556,-73.9797);
  var mapOptions = {
    center: myLatLong,
    zoom: 13
  };
  var map = new google.maps.Map(document.getElementById("map-canvas"),
      mapOptions);
  // To add the marker to the map, use the 'map' property
  var marker = new google.maps.Marker({
      position: myLatLong,
      map: map,
      title:"123 Broadway"
  });

}
google.maps.event.addDomListener(window, 'load', initialize);
</script>
